Website which organises school related stuff

Requirements:
---------
* python 3.3
* tornado (for python3)
* jinja2 (for python3)
* sqlalchemy (for python3)

Installation
---------
    sudo apt-get install python3.3 python3-pip
    sudo pip3 install tornado
    sudo pip3 install Jinja2
    sudo pip3 install sqlalchemy

Running
---------
    python3 main.py
